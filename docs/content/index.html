Typografi #

Globalt

Generelt om typografi i Lexto

  • Font-Familie: 'Montserrat', 'sans-serif', 'Open Sans' - native font stack skal implementeres
  • Standard font-size: 16px
  • Standard font-color: #1a1a1a
  • Side-titler bruker CTA farge
  • Headinger skal skrives semantisk

Ikoner #

I Lexto bruker vi FontAwesome Pro for å hente ulike SVG ikoner.

Retningslinjer:

  • Det skal kun brukest light ikoner med mindre annet er ønsket
  • Der er ikke satt noen regler på størrelse. Tilpass etter behov, men default holder som oftest
  • Brukes kun på logiske plasser, f.eks i navigasjon for å illustrere side-funksjon
  • Unngå overdrevent bruk av ikoner i knapper
  • Ved bruk av ikoner i knapper, skal de være høyrejustert

Eksempel på bruk av ikoner fra FA:

                        

                            Markup
                            <i class="fal fa-trash-alt"></>
                        

                        

                            Resultat

FontAwesome hentes inn via ekstern CDN. Dette skal etterhvert legges inn lokalt for å unngå risiko for feil dersom ekstern CDN skulle gå ned.

Bla gjennom de ulike ikonene her:

https://fontawesome.com/

Kjøreregler:

  • Hover farge skal være mørkere enn visningsfargen
  • Primary oransje (CTA) skal brukes med skjønn. Kun for å fremheve viktige elementer
  • Primary skal generelt brukes for der vi ønsker å føre bruker
  • Secondary skal brukes på f.eks avbryt knapper og valg brukeren har som skal være der, men ikke er hovedfunksjon

Globale Variabler #

Globale variabler (SCSS Variabler) er satt opp for blant annet farger. Alle variabler kan du finne under:

Lexto - Frontend\css\diverse\_variables.scss

Eksempel på tilgjengelige variabler

Variabel-navn Variabel-verdi
$color-cta #ff520d
$color-cta-hover #ee571b
$transition-default all 0.2s ease-in-out
$shadow-default 0px 1px 4px rgba(0, 0, 0, 0.25)
$border-radius-default 5px
